Description:
    Script to learn the transition matrix out of a FASTA file.

Usage:
    transition_matrix.sh  [options]  -- FASTA_FILE

Options:
    -h|--help           Help.
    -d|--outdir         Output directory. Default: $PWD.
    -t|--threads        Threads used. Default: 2.
    -k|--kmer_size      K-mer size used. Default: 2.

Examples:
    transition_matrix.sh -t 4 -k 1 -d out -- test.fa

Dependancies:
    transition_matrix.pl
    perl{Algorithm::Combinatorics}
